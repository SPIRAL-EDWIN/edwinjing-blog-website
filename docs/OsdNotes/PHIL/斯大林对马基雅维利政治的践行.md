# 斯大林对马基雅维利政治的践行

斯大林在苏联的统治手段，几乎可以说是切萨雷·波吉亚那个历史案例的“现代终极放大版”。他之所以与马基雅维利深度绑定，主要体现在以下几个历史事实上：

**1. 熟读并批注《君主论》** 历史档案表明，斯大林的私人图书馆里确实藏有马基雅维利的《君主论》，并且他在书上做了大量的眉批和标记。他极度认同书中“被人畏惧胜于被人爱戴”、“必须如狐狸般狡猾，如狮子般凶猛”的现实主义政治理念。

**2. “大清洗”（The Great Purge）与消灭政敌** 在1930年代，为了建立绝对的个人独裁，斯大林发动了残酷的“大清洗”。他以极其冷酷的政治手腕，通过构陷、伪造证据和严刑拷打，肉体消灭了几乎所有可能威胁他权力的布尔什维克建党元老（如托洛茨基、季诺维也夫、加米涅夫、布哈林）以及大量的高级将领。这完美契合了马基雅维利关于**“统治者在政治中不需要道德，必须无情铲除潜在威胁”**的教诲。

**3. 完美复刻波吉亚的“杀手套”策略（最经典的马基雅维利手段）** 斯大林在执行“大清洗”时，完美复刻了资料中提到的切萨雷·波吉亚处死自己将军的手段。斯大林先后启用了雅戈达（Yagoda）和叶若夫（Yezhov）作为内务人民委员部（秘密警察，即NKVD）的首脑，让他们替自己执行最血腥的逮捕和枪决。 当这两人双手沾满鲜血、造成极大的社会恐慌和民怨后，斯大林便以“滥用职权、迫害无辜”的罪名将他们逮捕并秘密处决，随后换上贝利亚（Beria）。通过这种方式，**斯大林巧妙地将所有残暴的黑锅推给了执行者，自己则继续在公众面前维持着“仁慈的伟大领袖”的形象**。这与马基雅维利“统治者必须注重公众形象（Public appearance is everything）”以及“巧妙利用残忍”的策略如出一辙。

**总结来说：** 斯大林被视作马基雅维利式的暴君，并不是因为他像个疯子一样为了杀人而杀人（马基雅维利也反对无意义的暴政），而是因为他**将国家视为一台必须被绝对控制的机器，将道德完全剥离于政治之外（情境伦理学），极其冷静、冷酷且精于算计地使用恐怖手段来巩固个人的绝对统治**。这就是他为什么被当成马基雅维利主义的最高实操模板的原因。
