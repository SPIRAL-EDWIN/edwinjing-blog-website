# 纯粹的大众民主极易变成暴民政治

**纯粹的大众民主最容易堕落为暴乱（暴民政治）** 这一观点，在课程中主要源自神学家约翰·加尔文（John Calvin）对卢梭（Rousseau）政治哲学的深刻影响

- 可以从以下四个层面来深刻理解：
    1. **现实操作的不可行性（直接民主的低效）** 卢梭语境下的“纯粹民主制”指的是所有公民直接参与所有决策的“直接民主” 。老师在课上举例说，如果在现代社会修一座桥都需要全体公民进行公投，这在现实操作中是根本行不通的。
    2. **“公意”与个体自私欲望的致命冲突** 纯粹的大众民主制建立在一个危险的假设上：即全民投票的结果就能自然而然地代表旨在实现公共利益的“公意（general will）” 。然而，个体在社会中的意志往往倾向于以自我为中心（egocentric）。全民投票得出的通常只是所有个人自私欲望的“总和”，这往往与真正符合国家长远利益的“公意”背道而驰。
    3. **选民缺乏绝对的理性与无私** 除非社会中的**每一个个体**都拥有极高的素养、掌握充分的信息，并且能够做出完全理性、非自我中心的选择，大众民主才可能等同于公意。但经验告诉我们，即使在高度文明和发达的社会中，大众投票也几乎不可能摆脱党派偏见和结党营私（factions）的干扰，选民极易做出盲目和冲动的决定。
    4. **多数人的暴政对“法治”与人权的破坏** 在大众民主下，如果将“人民的意志”无限拔高并凌驾于法律之上，就会与“法治（Rule of Law）”产生致命的冲突 。老师以瑞士的公投制度为例指出，如果民众通过公投做出了违宪或侵犯少数族裔权利的决定（例如剥夺有色人种的投票权），政府也必须强制执行这种所谓的“人民意志”。在这种情况下，哪怕是打着“公共利益”旗号的非自我中心决定，也可能合法地践踏人类的基本尊严和权利 。当民主失去宪法和法治的绝对制衡，宪法变得极度脆弱时，所谓的大众统治就会直接变为践踏少数人的“暴民政治” 。
- 正因如此，卢梭虽然反对将权力集中于一人的君主制，但他同样认为大众直接统治的民主制极易沦为暴乱。他最终推崇的是**由少数兼具智慧与道德的精英进行统治的贵族制（即代议制或议会共和制）**，认为这才是代表公意和实现公共利益的最佳政府形式 。
